##### **Using SageMaker hosted LLM as FM in BedRock**



1\.	In SageMaker Studio, choose JumpStart in the navigation pane. Choose the Bedrock ready Hugging Face model card. Deploy the model by choosing Deploy. 



2\.	Under Deployments in the navigation pane, choose Endpoints to view your available endpoints. 



3\.	Choose Use with Bedrock to start the registration process. You will be redirected to the Amazon Bedrock console. On the Register endpoint page, the SageMaker endpoint Amazon Resource Name (ARN) and model ARN have already been prepopulated. Review these values and choose Register.



4\.	In the navigation pane on the Amazon Bedrock console, choose Marketplace deployments under Foundation models. From the list of managed deployments, select your registered model, then choose Open in playground. Enter the prompt into the playground, then choose Run.
